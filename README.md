revert.css
===

Restore Browser/Spec default CSS styles

revert.css (repository) also includes the default CSS styles from:

1. W3C HTML4 spec
2. Mozilla Firefox
3. Chrome (webkit)
4. Opera
5. Internet Explorer 6-9+

Created as a result of: http://stackoverflow.com/questions/22509640/

Update:
